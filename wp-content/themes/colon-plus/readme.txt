Colon Plus is a MULTI-PURPOSE WordPress theme with a simple and modern design. It's perfectly RESPONSIVE and supports the RTL language. Theme can be used for business, hotels, travel, gym, sports, fitness, and fashion-related websites. The theme is SEO-friendly and TRANSLATION ready too. == Frequently Asked Questions ==

= Installation =
	
1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Colon Plus in the search form and press the 'Enter' key on your keyboard.
3. Click on 'Install' button to install and 'Activate' button to use.
